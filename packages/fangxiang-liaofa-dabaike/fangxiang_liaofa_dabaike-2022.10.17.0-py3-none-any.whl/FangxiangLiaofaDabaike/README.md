# 芳香疗法大百科

## 下载

### Docker

```
docker pull apachecn0/fangxiang-liaofa-dabaike
docker run -tid -p <port>:80 apachecn0/fangxiang-liaofa-dabaike
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install fangxiang-liaofa-dabaike
fangxiang-liaofa-dabaike <port>
# 访问 http://localhost:{port} 查看文档
```

```
# 访问 http://localhost:{port} 查看文档
```
