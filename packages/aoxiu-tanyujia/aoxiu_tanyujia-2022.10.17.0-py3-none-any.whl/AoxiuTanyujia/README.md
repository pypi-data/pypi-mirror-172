# 奥修谈瑜伽

## 下载

### Docker

```
docker pull apachecn0/aoxiu-tanyujia
docker run -tid -p <port>:80 apachecn0/aoxiu-tanyujia
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install aoxiu-tanyujia
aoxiu-tanyujia <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g aoxiu-tanyujia
aoxiu-tanyujia <port>
# 访问 http://localhost:{port} 查看文档
```
