# 图解阴阳师

## 下载

### Docker

```
docker pull apachecn0/tujie-yinyangshi
docker run -tid -p <port>:80 apachecn0/tujie-yinyangshi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install tujie-yinyangshi
tujie-yinyangshi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g tujie-yinyangshi
tujie-yinyangshi <port>
# 访问 http://localhost:{port} 查看文档
```
