## 下载

### Docker

```
docker pull apachecn0/qimen-dunjia-miji-quanshu-xia
docker run -tid -p <port>:80 apachecn0/qimen-dunjia-miji-quanshu-xia
# 访问 http://localhost:{port} 查看文档
```

