# 生命的游戏：禅塔罗手册

## 下载

### Docker

```
docker pull apachecn0/shengmingde-youxi-chantaluo-shouce
docker run -tid -p <port>:80 apachecn0/shengmingde-youxi-chantaluo-shouce
# 访问 http://localhost:{port} 查看文档
```
